# Simple-Calculator-Program

A clean modular calculator built with Python. This project was designed to explore **Structural Pattern Matching** and modular function design.

## 🚀 Features
* **Modern Syntax** Uses Python 3.10+ 'match-case' statements for cleaner logic compared to the traditional 'if-elif' chains.
* **Error Handling** Includes basic protection against division by zero and invalid numeric inputs.
* **Persistent Loop** Allows for multiple calculations without the need to restart the script.

## 🛠️ Tech Stack
* **Language** Python 3.10+
* **Paradigm** Functional/Procedural

## 📖 What I Learned
* Implementing 'Match-Case' in Python (Structural Pattern Matching).
* Handling user input and type conversion safely using 'try-except'.
* Using `if __name__ == "__main__"` as an entry point to manage script execution.
